# Amazon FBM 商品售价及利润计算公式

## 用途
计算亚马逊 FBM（卖家自发货）商品的建议售价和预期利润，确保利润不低于 $10。

## 变量定义
- **P** = 建议零售价（建议亚马逊售价）
- **G** = 货值总计
- **T** = 预估总额，含运费（大建云仓成本包含邮费）
- **C** = 综合成本

## 已知参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 退货率 | 10% | 10%的订单会退货 |
| 有效成交率 | 90% | 90%的订单成功成交 |
| 佣金 | 9% × P | 按有效成交额计算（10% × 90%×P） |
| 每单保险费 | $7 | 固定费用 |
| 优惠券折扣 | 10% × P | 按售价计算 |
| 优惠券扣点 | 2.5% × P | 按售价计算 |
| 广告费 | $2 | 固定费用 |
| 目标利润 | 5% × P | 或最低 $10 |
| 退货保险赔偿 | 60% × G | 退货时保险公司赔偿 |

## 综合成本
```
C = T + 9 - 0.06G
```
其中：
- **9** = 保险费 $7 + 广告费 $2
- **0.06G** = 退货率 10% × 保险赔偿 60% × 货值总计

## 核心公式

### 公式一：按售价 5% 利润计算
```
P1 = (T + 9 - 0.06G) ÷ 0.635
```
或：
```
P1 = C ÷ 0.635
```

### 公式二：当利润低于 $10 时，按最低利润 $10 计算
```
P2 = (T + 19 - 0.06G) ÷ 0.685
```
或：
```
P2 = (C + 10) ÷ 0.685
```

### 最终建议零售价
```
P = MAX(P1, P2)
```

**简要说明：**
- 第一个公式保证利润达到售价的 5%
- 第二个公式保证当 5% 利润不足 $10 时，售价会被重新拉高到至少赚 $10
- 最终建议零售价取两个公式算出来的较高值

## 利润公式
```
利润 = P × 0.685 - T - 9 + 0.06G
```

## 公式推导

### 收入比例计算
```
收入比例 = 有效成交率 - 佣金 - 优惠券折扣 - 优惠券扣点
收入比例 = 90% - 9% - 10% - 2.5% = 68.5%
```

**关键：佣金按有效成交额计算**
- 佣金 = 10% × 有效成交额 = 10% × 90%×P = 9%×P

### 公式一推导（利润 = 5%×P）
```
利润 = P × 0.685 - T - 9 + 0.06G
5%×P = P × 0.685 - T - 9 + 0.06G
P × 0.685 - 0.05P = T + 9 - 0.06G
P × 0.635 = T + 9 - 0.06G
P = (T + 9 - 0.06G) ÷ 0.635
```

### 公式二推导（利润 = $10）
```
利润 = P × 0.685 - T - 9 + 0.06G
$10 = P × 0.685 - T - 9 + 0.06G
P × 0.685 = T + 9 - 0.06G + $10
P × 0.685 = T + 19 - 0.06G
P = (T + 19 - 0.06G) ÷ 0.685
```

## 计算示例

### 示例1：正常商品（利润 ≥ $10）

**输入：**
- 预估总额含运费 T = $275.97
- 货值总计 G = $200.66

**计算：**
```
P1 = (275.97 + 9 - 0.06×200.66) ÷ 0.635
P1 = (275.97 + 9 - 12.04) ÷ 0.635
P1 = 272.93 ÷ 0.635
P1 = $429.81

P2 = (275.97 + 19 - 12.04) ÷ 0.685
P2 = 282.93 ÷ 0.685
P2 = $413.04

P = MAX(429.81, 413.04) = $429.81
```

**验证利润：**
```
利润 = 429.81 × 0.685 - 275.97 - 9 + 12.04
利润 = 294.42 - 275.97 - 9 + 12.04
利润 = $21.49 ✅
```

### 示例2：低利润商品（利润 < $10）

**输入：**
- 预估总额含运费 T = $66.01
- 货值总计 G = $49

**计算：**
```
P1 = (66.01 + 9 - 0.06×49) ÷ 0.635
P1 = (66.01 + 9 - 2.94) ÷ 0.635
P1 = 72.07 ÷ 0.635
P1 = $113.50

P2 = (66.01 + 19 - 2.94) ÷ 0.685
P2 = 82.07 ÷ 0.685
P2 = $119.81

P = MAX(113.50, 119.81) = $119.81
```

**验证利润：**
```
利润 = 119.81 × 0.685 - 66.01 - 9 + 2.94
利润 = 82.07 - 66.01 - 9 + 2.94
利润 = $10.00 ✅
```

## 注意事项

1. **货值或成本为0的商品**：无法计算，跳过处理
2. **公式假设**：退货率10%、保险赔偿60%是固定参数
3. **佣金计算**：按有效成交额（90%×P）计算，不是按售价
4. **最低利润**：确保每单利润不低于 $10

## 参数调整

如果参数变化，重新推导公式：

```python
# 可调参数
RETURN_RATE = 0.10       # 退货率
EFFECTIVE_RATE = 0.90    # 有效成交率
COMMISSION_RATE = 0.10   # 佣金率（按成交额）
INSURANCE_FEE = 7        # 每单保险费
COUPON_DISCOUNT = 0.10   # 优惠券折扣
COUPON_CLAWBACK = 0.025  # 优惠券扣点
AD_FEE = 2               # 广告费
TARGET_PROFIT_RATE = 0.05 # 目标利润率
RETURN_INSURANCE = 0.60  # 退货保险赔偿比例
MIN_PROFIT = 10          # 最低利润

# 收入比例
income_ratio = EFFECTIVE_RATE - COMMISSION_RATE * EFFECTIVE_RATE - COUPON_DISCOUNT - COUPON_CLAWBACK

# 公式一分母（利润按比例）
denominator_1 = income_ratio - TARGET_PROFIT_RATE

# 公式二分母（利润固定）
denominator_2 = income_ratio
```
